## 프로젝트 개요

본 프로젝트는 **부산광역시 1365 스마트시티 디지털트윈 플랫폼** 내 "도심 인공조명 생활안전" 서비스의 프론트엔드 구현 저장소입니다.

도시의 과도한 인공조명(빛공해)이 시민 생활에 미치는 영향을 분석·시뮬레이션하여,
행정기관과 시민 모두가 **데이터 기반으로 조명 환경을 이해하고 개선 방향을 도출**할 수 있도록 지원합니다.

> 🔗 **라이브 서비스 확인**: [https://1365twin.busan.kr/twin/map](https://1365twin.busan.kr/twin/map) <br/>
> 🔗 **서비스 소개 페이지**: [https://1365twin.busan.kr/twin/service/light](https://1365twin.busan.kr/twin/service/light)

## 서비스 배경

현대 도시의 야간 인공조명은 에너지 낭비와 생태계 교란을 유발하는 동시에,
보행자 안전과 생활 환경에도 직접적인 영향을 미칩니다.

부산광역시는 이를 해결하기 위해 **디지털트윈 기술**을 활용한 인공조명 생활안전 플랫폼을 구축하였으며,
시범 구역(부산진구)을 시작으로 **2026년까지 부산시 15개 구·군 전역으로 확산** 예정입니다.

## 주요 서비스 기능

### 대국민 서비스 (시민 대상)

| 기능 | 설명 |
|------|------|
| **영향분포 히트맵** | 도심 내 인공조명의 빛 영향 범위를 히트맵으로 시각화하여 빛공해 현황을 직관적으로 파악 |
| **광원수정 시뮬레이션** | 특정 광원의 방향·세기·색온도를 가상으로 수정하고 변화 결과를 실시간 시뮬레이션 |
| **핫스팟 진단분석** | 빛공해가 집중된 핫스팟 지역을 자동 탐지하여 우선 개선 대상지 시각화 |
| **조명환경관리구역 조회** | 법적으로 지정된 조명환경관리구역 경계 및 규제 기준 정보 조회 |

### 행정 서비스 (관리자 대상)

| 기능 | 설명 |
|------|------|
| **인공조명 정보조회** | 구역별 설치된 인공조명의 위치·종류·조도 등 상세 속성 정보 조회 |
| **차광막설치 시뮬레이션** | 차광막 설치 후 빛 차단 효과를 사전 시뮬레이션하여 설치 타당성 검토 |
| **개선 우선순위 선정** | 데이터 기반 분석으로 조명 환경 개선이 시급한 지역의 우선순위 자동 산출 |
| **구역별 조도 수준 진단** | 구역별 야간조도 측정값과 기준치를 비교하여 초과 여부 진단 |
| **보행 안전 동선 확인** | 야간 보행 시 조도 부족으로 위험한 동선을 식별하고 개선 필요 구간 도출 |
| **개선안 시뮬레이션 비교** | 복수의 조명 개선 시나리오를 생성하고 효과를 비교 분석 |
| **데이터 기반 개선 결정 지원** | 조도 측정 데이터와 시뮬레이션 결과를 종합하여 의사결정 지원 리포트 제공 |
| **야간조도 현황 파악** | 실시간 및 과거 야간조도 데이터를 기반으로 도시 전체 조명 현황 모니터링 |
| **합리적 개선 우선순위 선정** | 비용 대비 효과, 민원 빈도, 조도 초과율을 종합한 합리적 개선 계획 수립 |

## 디지털트윈 기반 시각화

본 서비스는 **3D 디지털트윈 맵** 위에서 모든 기능이 구현됩니다.

- 실제 부산시 도시 구조물과 건물을 3D로 모델링한 공간 위에서 인공조명 데이터를 오버레이
- 조명 영향 범위를 공간적으로 직관 파악 가능
- 히트맵, 핫스팟, 차광막 효과 등 모든 분석 결과를 **실시간 3D 뷰**로 확인

> 실제 디지털트윈 맵 체험: [https://1365twin.busan.kr/twin/map](https://1365twin.busan.kr/twin/map)

## 기술 스택

```
Frontend
├── 3D 디지털트윈 렌더링 (도시 공간 시각화)
├── WebGL 기반 히트맵 및 데이터 오버레이
├── 실시간 시뮬레이션 UI (광원수정, 차광막, 개선안 비교)
└── 반응형 대시보드 (대국민 / 행정 구분 인터페이스)

Data Integration
├── 야간조도 측정 데이터 API 연동
├── 인공조명 시설물 공간 데이터 처리
└── 조명환경관리구역 GIS 데이터 렌더링
```
